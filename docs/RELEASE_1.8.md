# Asgard TV 1.8 — Sources Manager

## Added

- Sources Manager UI.
- Add source template button.
- Edit and save `sources.txt` from Web UI.
- Reset saved sources to default assets.
- Sources search screen.
- Sources diagnostics screen.
- Source parser validation:
  - 8-column format
  - supported source type
  - required `{query}` for `search_template`
- Version updated to 1.8.0.
- Backlog added for:
  - remote + touch screen support
  - GitHub update system

## Notes

This release does not add bundled questionable catalogs, mirror search, DRM bypass, Cloudflare bypass or captcha bypass.
